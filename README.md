# DPmaps

Interactive map to draw Roadworks & Utilities

## Features
- Toggle layers: cadastre, contours and stormwater.
- Draw layers: roadworks & utilities.
- Search Job ID in Search Bar.
- Export to Excel for easier tracking of jobs.

Cadastre and Contour have been loaded as a Release (instead of Repository) then pulled from Cloudflare due to file size limit on Github.
